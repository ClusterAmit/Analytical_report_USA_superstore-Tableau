Analytical_report_USA_superstore-Tableau
Project Overview

This project features an interactive analytical dashboard developed in Tableau using the USA Superstore dataset. The objective is to analyze retail performance and convert raw sales data into actionable business insights.

Key Objectives

Analyze sales, profit, quantity, and discount trends

Identify high- and low-performing regions, categories, and customer segments

Support data-driven decision-making through visual insights

Key Features

Interactive dashboards with dynamic filters

KPI-focused visualizations for quick performance evaluation

Trend analysis across time, geography, and product categories

Tools & Technologies

Tableau – Data visualization and dashboard creation

USA Superstore Dataset – Retail sales data

Insights Covered

Regional and state-wise performance

Category and sub-category profitability

Customer segment analysis

Impact of discounts on profit

Use Case

This project is suitable for business analysts, data analysts, and beginners looking to understand retail analytics and Tableau dashboarding through a real-world dataset.
